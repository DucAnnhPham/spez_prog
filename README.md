## Projekt: Müllklassifikation in Deutschland

### 1. Executive Summary – Kurze Zusammenfassung des Projekts.
    Das Projekt entwickelt eine containerisierte AI-Anwendung zur Müllklassifikation in Deutschland. 
    Die Kernkomponente ist ein KI-Service, der über die OpenAI API Objekte analysiert und zuverlässig einer Müllkategorie zuordnet.
    Bei Unsicherheit gibt die AI bewusst „Ich weiß nicht“ aus, um Halluzinationen zu vermeiden.
   
### 2. Ziele des Projekts – Welche Ziele verfolgt Ihr Projekt, welches Problem wird gelöst?  
    Ziel dieses Projekts ist es, die Mülltrennung in Deutschland zu verbessern, indem eine AI-gestützte Lösung bereitgestellt wird, die Nutzern hilft, Abfälle korrekt zu klassifizieren.
    So können Nutzer Müll identifizieren und entsprechend entsorgen, was zu einer effizienteren Mülltrennung und einem positiveren Umwelteinfluss beiträgt.
    Damit wird ein Beitrag zur Reduzierung von Umweltverschmutzung und zur Förderung nachhaltiger Praktiken geleistet.
